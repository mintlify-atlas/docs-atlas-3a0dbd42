  <div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
    <h2 className="text-2xl sm:text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
    <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get DBHub running in minutes with Docker, npm, or demo mode.</p>
    
    <Steps>
      <Step title="Run DBHub with Docker">
        Launch DBHub with your database connection string:
        
        ```bash Docker
        docker run --rm --init \
          --name dbhub \
          --publish 8080:8080 \
          bytebase/dbhub \
          --transport http \
          --port 8080 \
          --dsn "postgres://user:password@localhost:5432/dbname?sslmode=disable"
        ```
        
        <Note>Replace the DSN with your actual database connection details. DBHub supports PostgreSQL, MySQL, MariaDB, SQL Server, and SQLite.</Note>
      </Step>

      <Step title="Or use npm to run DBHub">
        No Docker? Run directly with npx:
        
        ```bash npm
        npx @bytebase/dbhub@latest \
          --transport http \
          --port 8080 \
          --dsn "postgres://user:password@localhost:5432/dbname?sslmode=disable"
        ```
      </Step>

      <Step title="Try demo mode (no database required)">
        Explore DBHub with a built-in SQLite employee database:
        
        ```bash Demo Mode
        npx @bytebase/dbhub@latest --transport http --port 8080 --demo
        ```
        
        <Info>Demo mode is perfect for testing DBHub's capabilities without connecting to a real database.</Info>
      </Step>

      <Step title="Access the workbench">
        Open your browser to `http://localhost:8080` to access the built-in web workbench. Execute queries, explore schemas, and test MCP tools visually.
        
        <Tip>The workbench provides a visual interface for all DBHub tools without requiring an MCP client.</Tip>
      </Step>
    </Steps>
